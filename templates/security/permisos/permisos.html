Información Importante

El sistema de permisos permite un control granular sobre las funcionalidades del sistema. Asegúrate de configurar correctamente los permisos para mantener la seguridad.

  • • Los permisos se asignan por grupo para facilitar la administración
  • • Cada módulo puede tener múltiples permisos específicos
  • • Los usuarios heredan los permisos de sus grupos asignados
  • • Los cambios en permisos se aplican inmediatamente
